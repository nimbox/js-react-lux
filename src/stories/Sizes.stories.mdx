# Sizes

We are using a basic font size for `html` of `14px`.  Most sites use `16px` as
the default font size, but since our applications are very data intensive we
need to pack more density. We tried `12px` but the smaller sizes were really to
small.

When applying the default scalings from tailwind we end up with these sizes. We
are only using `rems` as units so `px` are only for reference.

# Paddings

Each size requires different paddings to make sure that the space around the
text looks ok. Buttons, Inputs, and other elements should use these defaults for
their paddings. In fact to standardize this we will use the `paddings[]` object
that given a size (`xs`, `sm`, `base`, `lg`) provides the font size and  padding
classes.

We are using these defaults:
