# Avatar

Avatars are representations of people. They can be initials with colors or an
image.

<Canvas>
    <Story name="Basic">
        <div className="bg-gray-200"><Avatar initials="KE" color="#ecf0f1" backgroundColor="#e74c3c"/>&nbsp;Katniss Everdeen</div>
    </Story>
</Canvas>

If only one color is provided it is assumed to be the background color and
the foreground color is choosen as white or black depending on the luminosity
of the background.

<Canvas>
    <Story name="One Color">
        <div><Avatar initials="KE" color="#e74c3c"/>&nbsp;Katniss Everdeen</div>
        <div><Avatar initials="KE" color="#f1c40f"/>&nbsp;Katniss Everdeen</div>
    </Story>
</Canvas>

If an image is provided then the initials, color and background color are
ommitted. And the image is placed instead.

<Canvas>
    <Story name="Image">

    </Story>
</Canvas>
