# More Options Button

A regular pattern is to a use a show more button.

<Preview>
    <Story name="Show more link">
        {() => {
            const [show, setShow] = useState(false);
            return (
                <>
                <div>Before</div>
                <MoreOptionsButton value={show} onChange={(show) => setShow(show)}>
                    <div>Extra content</div>
                </MoreOptionsButton>
                <div>After</div>
                </>
            );
        }}
    </Story>
</Preview>

Change the size of the text for a less intrusive button.

<Preview>
    <Story name="Show more link small">

                    <div>Extra content</div>
                </MoreOptionsButton>

                </>
            );
        }}
    </Story>
</Preview>
