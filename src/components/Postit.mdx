# Postit

Postit notes are a great way to hightlight relevant information in a page. They
can expand multiple witdhts, but bare in mind that there is a minimum with and
height of `minWidth: '64px'` and `minHeight: '5rem'`. If the `Postit` is smaller
the shadow will show through.
