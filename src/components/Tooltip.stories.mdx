# Tooltip

There are two types of tooltips. 

## Simple

Simple tooltips are taken from `data` elements inside every element. Whenever
there is a `mouseover` (not `mouseenter` since it doesn't bubble up) event on
any element with a `data-tooltip` attribute, the tooltip will be shown. This is
usefull for showing simple and short one liners.

These tooltips are placed using the same `Popper` engine used for posititiong
all other elements.

<Canvas>
    <Story name="Basic">
        <div className="inline-block px-3 py-2 border rounded bg-red-100" data-tooltip="Hover text">Hover over me</div>
    </Story>
</Canvas>

When applying to different text sizes it positions accordingly

<Canvas>
    <Story name="Applied to different sizes">

            <span className="text-sm bg-red-100" data-tooltip="Hover text" data-tooltip-show>Small</span>

            <span className="text-lg bg-red-100" data-tooltip="Hover text" data-tooltip-show>Large</span>
            <span className="text-xl bg-red-100" data-tooltip="Hover text" data-tooltip-show>Extra Large</span>
        </div>
    </Story>
</Canvas>

It even works on svg!

## Complex

Complex tooltips are the ones with more elaborated than one liners and that
require more formatting. They are rendered exactly with the same display
characteristics, but are handled by the application and not the browser.

This is no different than `popover`.
